What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It tells the browser how to display text, images, links, videos, and layout on a webpage.

Why Use HTML?

  • To structure content (like headings, paragraphs, images)
  • To create forms, tables, navigation menus, and more
  • It works with CSS (for styling) and JavaScript (for interaction) to build full websites
  • HTML is the foundation of all web development

Important concepts to know about HTML

    1.Tags

  • A tag is the code that tells the browser what to do or display.
  • Tags are written using angle brackets < >.

    • Types of tags:

  • Opening tag: <p>
  • Closing tag: </p>
  • Self-closing tag: < img />, < br />, < hr />
    • Example:

    <p Hello, world! /p>

    2. Element

  • An element is the complete structure from opening tag to closing tag including content.
  • It represents a piece of the document.
    • Example:

    <p Hello, world! /p>

    This entire thing is a paragraph element:

  • Tag: < p > </p>
  • Content: Hello, world!
  • Element: < p > Hello, world! </p>

    • ✅ A self-closing tag like is also an element — just without inner content.

    3. Attribute

  • An attribute provides additional information about an element.
  • It's written inside the opening tag.
  • Syntax: name="value"
    • Example:

    < img src="logo.png" alt="Company Logo" width="200">

    src, alt, and width are attributes of the element.

    They give details like the image file path, alternative text, and width.

    ✅ Summary

    Term Definition Example
    Tag HTML keyword inside angle brackets <h1>, </p>, <img />
    Element Full structure with tag and content <h1> HEADING 1 </h1>, <img />
    Attribute Adds properties or behavior to tags href="", src="", id=""
